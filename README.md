# RentPilot Frontend

This is the frontend for the RentPilot application. It is built using React and Tailwind CSS.

## Getting Started

To get started, you'll need to have Node.js and npm installed on your machine. You can download them from the official Node.js website.

Once you have Node.js and npm installed, you can install the project dependencies by running the following command in your terminal:

```
npm install
```

After the dependencies are installed, you can start the development server by running the following command:

```
npm start
```

This will start the development server and open your application in your default web browser. You can now start building your application and making changes to the code.
